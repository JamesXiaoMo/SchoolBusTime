## 简介
“校车时间”是一个用来给日本广岛县福山大学的学生实时查询下一班校车的程序。
灵感源自于一个晴朗的周五午后，即使我身处人工智能课的教室，但浮躁的内心却早已归心似箭。
此时我打开了福山大学的官网查询下一班校车出发的时间。
此时我意识到一个简单的查询，却需要在官网上点击数次，这显然不符合人机交互的原则。
于是“校车时间”这一灵感就应运而生。

在设计之初我认为“校车时间”应该是一个这样的应用：
* 快速查询
* 安全
* 简单易用

于是以此为目标，“校车时间”诞生了。
